# IT3220 – Buổi 2 (Lab02) – Ví dụ thực hành PHP cơ bản
Ngày tạo: 2025-12-30

## Chạy nhanh (XAMPP)
1) Start Apache trong XAMPP  
2) Copy thư mục `lab02` vào: `C:\xampp\htdocs\`  
3) Mở: http://localhost/lab02/

## File
- index.php: menu
- 01_variables.php: biến + echo/print
- 02_types.php: kiểu dữ liệu + var_dump/gettype/is_*
- 03_operators_math.php: phép toán số học + printf/number_format
- 04_strings.php: nối chuỗi + nháy đơn/kép
- 05_constants.php: hằng define/const
- 06_calc_get.php: máy tính GET
- 07_electric_bill_get.php: tính tiền điện GET (3 bậc minh họa)
- 08_average_grade_get.php: tính điểm TB GET

Gợi ý: SV nộp zip `lab02` + ảnh chụp chạy được URL mẫu.
